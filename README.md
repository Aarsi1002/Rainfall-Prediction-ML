# Rainfall-Prediction-ML
Machine learning project for rainfall prediction
Rainfall Prediction using Machine Learning

This project applies machine learning models (XGBoost, Random Forest, Decision Tree) to predict rainfall patterns using climate data.

Key Features:
- Data preprocessing and feature engineering
- Model training and evaluation
- Achieved ~25% improvement in accuracy over baseline models

Tools Used:
Python, Pandas, Scikit-learn, XGBoost

This project demonstrates an end-to-end machine learning workflow from data preparation to model evaluation.
